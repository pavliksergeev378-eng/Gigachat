GigaChat Diagnostics

Страница не требует вставки команд в Console. Полный token не выводится, только первые 8 символов.

1. Проверка загрузки JS

Нажми кнопку.

2. Текущая авторизация

Нажми кнопку.

3. Group Sync

Нажми кнопку.

4. Тестовое создание группы

После создания нажми «Мои группы» и «Список групп».

Нажми кнопку только если можно создать тестовую группу.

5. Вступить в группу

Нажми кнопку только если есть group_id.
